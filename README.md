BFS(Graph, Start)

Create an empty queue Q
Create an empty set VISITED

Mark Start as visited
Insert Start into Q

While Q is not empty
    Remove a vertex u from Q
    Print u

    For each vertex v adjacent to u
        If v is not in VISITED
            Mark v as visited
            Insert v into Q
        End If
    End For
End While
